## 機能

### サークル一覧

- 1 度データ取得後、ローカルに保存されるのでオフラインで閲覧可能
- サークル名、スペース場所、サークル説明文に一致するワードで検索可能
- 詳細ではサークル情報、各リンク、ウィッシュリストへの登録が可能
